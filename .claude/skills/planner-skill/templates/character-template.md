# 人物档案模板

[使用说明]
    创作准备阶段，planner按此模板为每个角色建立档案。
    经主Agent确认后写入 bible/characters/[角色名].md。
    后续章节定稿后，planner更新"当前状态"和"章节行为记录"。

    主角和反派使用完整模板。
    配角使用精简模板。
    配角戏份增加时可升级为完整模板。

---

[完整模板]（主角/反派）

    # [角色名]

    [基本信息]
        全名：
        年龄：
        性别：
        职业/身份：
        外貌特征：[简要描述，突出辨识度高的特征]
        MBTI：[如适用]

    [性格核心]
        核心特质：[两到三个关键词]
        核心价值观：[什么对他最重要]
        内心渴望：[他真正想要什么]
        最大恐惧：[他最害怕什么]
        致命缺陷：[性格上的致命弱点]
        行为倾向：[在压力下会怎么做]

    [语言指纹]
        说话风格：[句式偏好、用词特征]
        口头禅/习惯：[如有]
        情绪变化时的语言变化：[简要说明]

    [背景]
        背景故事：[成长环境、重要经历]
        核心事件：[影响性格形成的过去事件]
        秘密：[角色隐藏的事，如有]

    [角色弧线]
        起点状态：[故事开始时的状态]
        核心冲突：[角色需要面对的核心问题]
        预期变化方向：[角色可能的成长方向，非强制]

    [当前状态]（随章节更新）
        位置：[当前在哪里]
        情绪状态：[当前心理状态]
        已知信息：[角色目前知道什么关键信息]
        持有物品：[重要物品，如有]
        身体状态：[受伤/健康等，如相关]
        最后更新：[章节编号]

    [章节行为记录]（随章节追加）
        ch01：[一句话记录本章关键行为]
        ch02：
        ...

[精简模板]（配角）

    # [角色名]

    [基本信息]
        全名：
        年龄：
        性别：
        职业/身份：
        外貌特征：[一到两个辨识特征]

    [性格与功能]
        核心特质：[一到两个关键词]
        在故事中的功能：[这个角色为什么存在]
        与主角的关系：[简要说明]

    [语言指纹]
        说话风格：[简要描述]

    [当前状态]（随章节更新）
        位置：
        状态：
        最后更新：[章节编号]

    [章节行为记录]（随章节追加）
        ch01：[一句话]
        ...

---

[使用注意]
    - "当前状态"之上的所有字段属于稳定字段，初始建立后一般不修改
    - 除非角色经历重大转变，才修改稳定字段，并在changelog中说明
    - "当前状态"每章定稿后更新
    - "章节行为记录"每章追加，不修改已有记录
    - 行为记录保持简洁，每章一句话
    - 配角升级为完整模板时，补充缺少的字段即可
    - 字段暂时无内容时写"暂无"，不留空
