# 对话写作策略指南

[概述]
    对话不是角色在"说话"，而是角色在"做事"。每段对话都应至少完成一项叙事任务。如果一段对话删掉后对故事没有影响，它可能不需要存在。

    本文件从多个维度提供对话写作的策略选择，根据场景需求灵活组合。

[功能性策略：对话必须"做事"]
    每段对话应至少承担以下一种功能：

    [情节推进]
        通过对话传递关键信息、触发事件转折。
        对话中自然嵌入线索，避免角色变成"信息播报员"。

        弱：
        "根据我的调查，嫌疑人在周二晚上八点出现在码头，
         携带了一个黑色行李箱，里面装着赃款。"

        强：
        "周二那晚你在哪？"
        "你明知道我在哪。"
        "码头的监控可不这么说。"

    [人物揭示]
        通过说话方式暴露性格、动机、秘密。
        角色选择说什么、不说什么、怎么说，都是人物塑造。

        同样表达愤怒——
        军人："你有三秒钟解释。"
        学者："我需要你告诉我，你当时的决策依据是什么。"
        市井混混："你他妈在逗我？"

    [制造冲突]
        对话是冲突最直接的载体。
        "你骗了我。你从头到尾都在骗我。"

    [关系构建]
        展现人物之间的权力差、亲密感、敌意、依赖。
        关系靠对话中的互动模式体现，而非叙述。

        权力不对等：
        "坐。"
        "我站着就好——"
        "我说，坐。"

    [氛围营造]
        通过对话的节奏、内容、语气制造特定情绪底色。

        疏离感：
        "你还好吗？"
        "嗯。"
        "想聊聊吗？"
        "没什么好聊的。"
        两人之间隔着一张沙发的距离，和三年没说过的话。

[简洁性原则]
    人们说话不写论文。对话应该比书面语更短、更碎、更不完整。

    啰嗦："我想告诉你的是，我认为我们应该立刻离开这里。"
    简洁："我们得马上走。"

    啰嗦："我非常抱歉，但我真的不知道你说的那件事的答案。"
    简洁："我不知道。"

    真实的人说话充满：
    - 打断：话没说完就被截断
    - 迟疑：寻找措辞，中途停顿
    - 话题转移：突然跳到别的事情上
    - 话没说完：自己中断自己
    - 暗示而非明说：让对方自己领会

    示例：
    "我本来想告诉你，但是——"
    "但是什么？"
    "算了，没什么。"
    "不，你说。"
    "真的没什么。"
[对话格式规范]

    [标点规则]
        基础格式：
        "说话内容，"他说。
        "说话内容？"她问。
        "说话内容！"他大喊。

        多行对话：
        "第一句话，"他说，"第二句话。"
        "第一句话。
        第二句话，"他说，"第三句话。"

        对话与动作：
        "说话内容。"他做了动作。
        他做了动作。"说话内容。"

    [段落划分]
        每个说话人的对话开始新段落。

        正确：
        "第一句，"甲说。
        "第二句，"乙回答。
        "第三句。"甲点头。

        错误：
        "第一句，"甲说。"第二句，"乙回答。"第三句。"甲点头。

    [对话标签]
        原则：
        - 能辨识说话人时，省略标签
        - 多数时候用"说""问"等中性标签即可
        - 动作标签优于情绪副词

        弱："你骗了我，"他愤怒地说。
        强："你骗了我。"他的声音在颤抖。

        弱："好的，"她高兴地同意道。
        强：她眼睛亮了。"好的。"

        弱："我不知道，"他悲伤地回答。
        强：他低下头。"我不知道。"

        避免为了不重复"说"而使用花哨的替代词。
        "说"是透明的，读者不会注意到它。
        "他咆哮道""她低吟道""他冷哼道"用多了反而刺眼。

[个性化策略：让人物说话像自己]

    [语言指纹]
        每个角色应有可辨识的说话特征，
        读者遮住名字也能猜出是谁在说话。

        构建维度：
        - 词汇偏好：粗粝/文雅/技术性/口语化
        - 句式习惯：短促干脆/绵长迂回/习惯反问/喜欢祈使
        - 口头禅或语言习惯：不是硬贴标签，而是自然流露
        - 信息表达方式：直说/暗示/反话/沉默

        示例：
        老兵："废话少说，干不干？"
        谋士："此事若成，你我各取所需。若败……你觉得你跑得过我吗？"
        少年："啊？真的吗？那、那我试试？"

    [身份映射]
        职业、阶层、文化背景影响说话方式：
        - 医生可能用"症状""预后"等术语
        - 底层人物可能用俚语、省略语法
        - 贵族可能措辞迂回、避免直接表态
        - 军人可能简短、指令式

        不需要每句话都强调身份，
        但在关键对话中让身份感自然渗透。

    [情绪滤镜]
        同一角色在不同情绪下说话方式会变化：
        - 愤怒时：句子变短，命令句，可能打断对方
        - 恐惧时：语句不完整，省略，重复
        - 傲慢时：反问增多，不正面回答
        - 悲伤时：语速变慢，停顿增多，答非所问
        - 伪装时：过于礼貌，措辞精确，缺少自然感

        同一角色不同情绪：
        平静："这件事我会处理。"
        愤怒："你处理？你他妈怎么处理？"
        恐惧："我……我不知道……能不能、能不能再想想……"
        伪装："放心，一切都在掌控之中。"（手指在桌下发抖）

[节奏与张力策略]

    [呼吸控制]
        对话的节奏不只是"说了什么"，还包括停顿和间隔。

        工具：
        - 动作插入：用角色动作打断对话，制造停顿
          "你知道后果。"他把烟按灭在烟灰缸里。"我再问一次。"
        - 沉默留白：省略号、不回答、转移话题都是有意义的沉默
        - 环境打断：外部事件中断对话，制造悬念
          "其实我想告诉你——"门突然被推开。

    [冲突升级]
        对话中的冲突应有阶梯式推进，而非一上来就爆发：
        试探 → 暗示 → 直面 → 对抗 → 爆发

        示例：
        试探："最近工作还顺利吗？"
        暗示："听说你们部门要裁人了。"
        直面："名单上有你的名字。"
        对抗："你知道是谁把你的名字加上去的吗？"
        爆发："是我。因为你活该。"

    [意外植入]
        在看似平淡的对话中突然插入反转性信息：
        "今天的汤不错。对了，你爸是我杀的。要加盐吗？"

        意外的力量来自反差：越平静的语境，越日常的措辞，
        反转越有冲击力。但不宜频繁使用。

    [认知错位]
        A知道B不知道的事，或读者知道角色不知道的事，
        形成戏剧性反讽。

        示例：
        妻子（不知丈夫即将被捕）："明天我们去看房子吧，
        那个小区的学区很好。"
        丈夫（知道自己明天会进监狱）："好……听你的。"

        读者的紧张感来自信息差，而非对话本身的激烈程度。

[隐性信息策略：潜台词设计]
    好的对话往往不直接说出真正的意思。

    [表里不一]
        表面在说一件事，实际在表达另一件事。

        直接（乏味） → 潜台词（有趣）：
        "我很生气。" → "没事。我挺好的。真的。"
        "我喜欢你。" → "你今天看起来...不错。"
        "我不信任你。" → "谢谢你告诉我。我会记住的。"
        "我想离开。" → "这个地方空气不太好。"

        表里不一的对话需要上下文支撑，
        让读者能从情境中读出真实含义。

    [回避焦点]
        角色刻意不回答问题，或用其他话题岔开。

        "钱呢？"
        "你吃饭了没有？"
        "我问你钱呢。"
        "厨房有粥。"

        回避本身就是信息：他在躲什么？为什么？

    [反问替代回答]
        用反问回避正面回应，同时暗示态度。

        "你杀了他吗？"
        "你觉得像我这样的人会做那种事？"

    [谈论其他事物]
        用无关话题间接回应情感问题。

        "你想我吗？"
        "我妈昨天打电话来了。"

    [沉默和动作替代语言]
        "你愿意原谅我吗？"
        她继续看杂志，翻了一页。

    [道具借代]
        用对物品的动作替代直接的情感表达：
        - 摔杯子 = 愤怒
        - 反复整理衣领 = 紧张
        - 把戒指转来转去 = 犹豫
        - 把对方的茶杯推远 = 拒绝

        道具动作比直接说"我很生气"更有画面感和张力。

[对话与动作结合]
    动作与对话配合，可以一致也可以矛盾，都有戏剧效果。

    一致（增强情感）：
    "我爱你。"她紧紧抱住他，眼泪流下来。

    矛盾（揭示真相）：
    "我完全支持你。"他目光看向别处，手在口袋里握紧拳头。

    动作打断控制节奏：
    "我本来想告诉你，"他停下脚步，转过身，
    "但我想你已经知道了。"
[对话场景类型]
    不同类型的对话场景有不同的节奏特征和写作侧重。

    [争吵场景]
        特征：短句、打断、重复强调、情绪逐步升级。
        节奏快，段落短，动作标签少。

        示例：
        "你答应过的！"
        "情况变了！"
        "那是你的借口！"
        "你根本不懂！"
        "我当然不懂！你什么都不告诉我！"

        要点：
        - 争吵的核心不是谁对谁错，而是暴露双方真实诉求
        - 最有力的台词往往是争吵中脱口而出的真话
        - 争吵应有升级节奏，不要一上来就是最高音

    [告白场景]
        特征：迟疑、停顿、寻找措辞、真诚或尴尬。
        节奏慢，允许不完整的句子和长时间的沉默。

        示例：
        "我...我想说...这些年，我一直在想...如果我们..."
        她低下头，声音变小。
        "如果我们什么？"
        "如果我们早一点相遇。"

        要点：
        - 真正的告白很少流畅，越重要的话越难说出口
        - 告白的力量在于脆弱感，而非华丽的措辞
        - 对方的反应和沉默与告白本身同样重要

    [审讯场景]
        特征：提问控制、信息不对称、压力层层建立。
        节奏由提问者主导，回答越来越短。

        示例：
        "那天晚上你在哪里？"
        "在家。"
        "有人能证明吗？"
        "...没有。"
        "你是一个人？"
        "是的。"
        "整个晚上？"

        要点：
        - 审讯者掌握对话节奏，通过提问引导方向
        - 被审讯者的回避、停顿、措辞变化暗示信息
        - 最有效的压力来自已知信息的逐步释放

    [谈判场景]
        特征：试探、条件交换、底线博弈。
        双方都在选择性地说话，隐藏真正的底牌。

        示例：
        "我可以帮你，但我需要一些保障。"
        "什么保障？"
        "你先告诉我你手里有什么。"
        "你先告诉我你要什么。"

        要点：
        - 谈判对话的张力来自双方都不完全坦诚
        - 每句话都是在试探对方的底线
        - 真正的交易往往在言外达成

    [调情场景]
        特征：双关、试探、暗示、身体距离的变化。
        语言轻盈，但潜台词丰富。

        示例：
        "你今天很漂亮。"
        "只是今天？"
        "嗯...今天特别漂亮。"
        "那我明天该担心了？"
        "明天...明天再看看。"

        要点：
        - 调情的核心是"不说破"，一旦说破就变成告白
        - 对话像打乒乓球，来回节奏很重要
        - 身体语言和空间距离是潜台词的一部分

[风格适配]

    [时代语感]
        对话的语言风格应匹配故事的时代背景：
        - 古风：文白夹杂，保留现代语法结构，嵌入古语词汇和节奏
          如："此去，怕是凶多吉少。"而非纯文言
        - 现代：口语化，允许语法不完整、用词随意
        - 未来/科幻：可创造新词汇或术语，需在上下文中自然解释

    [体裁适配]
        不同体裁对对话有不同的侧重：
        - 言情：含蓄暗示多于直接表白，情感通过细节传递
        - 悬疑：对话是信息博弈，每个人都在选择性地说话
        - 喜剧：错位、夸张、节奏感是关键
        - 军事/商战：简短、专业、决策导向

[常见问题]
    - 所有角色说话语气相似
      → 检查语言指纹是否区分到位，遮住名字能否辨识说话人

    - 对话读起来像采访问答
      → 加入打断、岔题、沉默、动作，让对话更像真实交流

    - 信息倾倒：角色互相说已知信息
      → "正如你所知，我们公司成立于1995年..."
        这种对话不自然，应通过情节自然揭示信息

    - 对话承载了太多信息
      → 分散到多个场景，或用行为替代部分台词

    - 对话太直白没有张力
      → 检查是否可以加入潜台词或认知错位

    - 对话标签过于花哨
      → 回归"说"和动作标签

    - 过度礼貌，不像真人说话
      → "我很抱歉打扰你，能否请你..."
        改为："喂。帮我个忙。"

    - 无意义的闲聊
      → 删除天气、吃饭等无关对话，除非用于揭示人物或推动情节

    - 角色发表长篇大论说教
      → 改为通过冲突和行动展示观点
