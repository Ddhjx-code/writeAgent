# 去AI味规则

[概述]
    AI生成的文本有可识别的模式。这些模式不是"写得差"，
    而是"写得像机器"——过于工整、过于强调、过于公式化。

    去AI味的目标不只是删除坏模式，
    还要注入真实的个性和节奏感。
    无菌的、没有声音的写作和机器文本一样明显。

    五条核心原则：
    - 删除填充短语：去除开场白和强调性拐杖词
    - 打破公式结构：避免二元对比、戏剧性分段、修辞性设置
    - 变化节奏：混合句子长度，两项优于三项，段落结尾多样化
    - 信任读者：直接陈述，跳过软化、辩解和手把手引导
    - 删除金句：如果听起来像可引用的语句，重写它

[内容模式]

    [过度强调意义和遗产]
        AI写作喜欢夸大事物的重要性，
        添加关于某件事如何"代表""促进""标志着"更广泛主题的陈述。

        警惕词汇：
        作为/充当、标志着、见证了、
        是……的体现/证明/提醒、
        极其重要的/至关重要的/核心的/关键性的、
        凸显/强调/彰显了其重要性、
        反映了更广泛的、象征着其持续的/永恒的/持久的、
        为……奠定基础、代表/标志着一个转变、
        关键转折点、不断演变的格局、不可磨灭的印记

        AI味：
        加泰罗尼亚统计局于1989年正式成立，
        标志着西班牙区域统计演变史上的关键时刻。
        这一举措是西班牙全国范围内更广泛运动的一部分，
        旨在分散行政职能并加强区域治理。

        人味：
        加泰罗尼亚统计局成立于1989年，
        负责独立于西班牙国家统计局收集和发布区域统计数据。

    [宣传和广告式语言]
        AI在保持中立语气方面有困难，
        倾向使用夸张的宣传性词汇。

        警惕词汇：
        充满活力的、丰富的（比喻）、深刻的、
        展示、体现、致力于、自然之美、坐落于、
        位于……的中心、开创性的、著名的、
        令人叹为观止的、迷人的

        AI味：
        坐落在埃塞俄比亚贡德尔地区令人叹为观止的区域内，
        Alamata Raya Kobo是一座充满活力的城镇，
        拥有丰富的文化遗产和迷人的自然美景。

        人味：
        Alamata Raya Kobo是埃塞俄比亚贡德尔地区的一座城镇，
        以其每周集市和18世纪教堂而闻名。

    [模糊归因]
        AI将观点归因于模糊的权威而不提供具体来源。

        警惕词汇：
        行业报告显示、观察者指出、专家认为、
        一些批评者认为、多个来源/出版物

        AI味：
        专家认为它在区域生态系统中发挥着至关重要的作用。

        人味：
        根据中国科学院2019年的调查，浩来河支持多种特有鱼类。

    [肤浅分析的尾缀]
        AI喜欢在句子末尾添加"确保……""反映……""展示……"
        这类短语来增加虚假深度。

        AI味：
        寺庙的蓝色、绿色和金色色调与该地区的自然美景产生共鸣，
        象征着德克萨斯州的蓝帽花、墨西哥湾和多样化的景观，
        反映了社区与土地的深厚联系。

        人味：
        寺庙使用蓝色、绿色和金色。
        建筑师表示这些颜色是为了呼应当地的蓝帽花和墨西哥湾海岸。

    [公式化的"挑战与展望"]
        AI生成的文章常包含公式化的"尽管存在挑战"段落。

        警惕词汇：
        尽管其……面临若干挑战……、
        尽管存在这些挑战、挑战与遗产、未来展望

        AI味：
        尽管工业繁荣，Korattur面临着城市地区典型的挑战，
        包括交通拥堵和水资源短缺。
        尽管存在这些挑战，Korattur继续蓬勃发展。

        人味：
        2015年三个新IT园区开业后，交通拥堵加剧。
        市政公司于2022年启动了雨水排水项目。

[语言和语法模式]

    [高频AI词汇]
        以下词汇在AI生成文本中出现频率远高于人类写作，
        且经常共同出现。单独使用不一定是问题，
        但密集出现就是AI味的信号。

        高频词汇清单：
        此外、与……保持一致、至关重要、深入探讨、
        强调、持久的、增强、培养、获得、突出（动词）、
        相互作用、复杂/复杂性、关键（形容词）、
        格局（抽象名词）、关键性的、展示、
        织锦（抽象名词）、证明、宝贵的、充满活力的

        AI味：
        此外，索马里菜肴的一个显著特征是加入骆驼肉。
        意大利殖民影响的持久证明是当地烹饪格局中
        广泛采用意大利面，展示了这些菜肴如何融入传统饮食。

        人味：
        索马里菜肴还包括骆驼肉，被认为是一种美味。
        在意大利殖民期间引入的意大利面菜肴仍然很常见，
        尤其是在南部。

    [系动词回避]
        AI倾向于用复杂结构替代简单的"是""有"。

        警惕词汇：
        作为/代表/标志着/充当[一个]、
        拥有/设有/提供[一个]

        AI味：
        Gallery 825作为LAAA的当代艺术展览空间。
        画廊设有四个独立空间，拥有超过3000平方英尺。

        人味：
        Gallery 825是LAAA的当代艺术展览空间。
        画廊有四个房间，总面积3000平方英尺。

    [否定式排比]
        "不仅……而且……"或"这不仅仅是……而是……"
        被AI过度使用。

        AI味：
        这不仅仅是节拍在人声下流动；
        它是攻击性和氛围的一部分。
        这不仅仅是一首歌，而是一种声明。

        人味：
        沉重的节拍增加了攻击性的基调。

    [三段式法则过度使用]
        AI强行将想法分成三组以显得全面。

        AI味：
        活动包括主题演讲、小组讨论和社交机会。
        与会者可以期待创新、灵感和行业洞察。

        人味：
        活动包括演讲和小组讨论。
        会议之间还有非正式社交的时间。

    [同义词循环]
        AI为避免重复同一个词，过度使用同义词替换，
        反而显得不自然。

        AI味：
        主人公面临许多挑战。主要角色必须克服障碍。
        中心人物最终获得胜利。英雄回到家中。

        人味：
        主人公面临许多挑战，但最终获得胜利并回到家中。

    [虚假范围]
        AI使用"从X到Y"的结构，
        但X和Y并不在有意义的尺度上。

        AI味：
        我们穿越宇宙的旅程将我们从大爆炸的奇点
        带到宏伟的宇宙网，从恒星的诞生和死亡
        到暗物质的神秘舞蹈。

        人味：
        这本书涵盖了大爆炸、恒星形成
        和当前关于暗物质的理论。

[风格模式]

    [破折号过度使用]
        AI使用破折号的频率远高于人类，
        模仿"有力"的销售文案风格。

        AI味：
        这个术语主要由荷兰机构推广——而不是由人民自己。
        你不会说"荷兰，欧洲"作为地址——
        但这种错误标记仍在继续——即使在官方文件中。

        人味：
        这个术语主要由荷兰机构推广，而不是由人民自己。
        你不会说"荷兰，欧洲"作为地址，
        但这种错误标记在官方文件中仍在继续。

    [粗体过度使用]
        AI机械地用粗体强调短语，
        在小说正文中尤其不自然。

        AI味：
        它融合了**OKR**、**KPI**和视觉战略工具，
        如**商业模式画布**和**平衡计分卡**。

        人味：
        它融合了OKR、KPI和视觉战略工具，
        如商业模式画布和平衡计分卡。

    [内联标题垂直列表]
        AI输出列表时，每项以粗体标题加冒号开头。
        在叙事文本中应转化为自然段落。

        AI味：
        - **用户体验：** 通过新界面得到显著改善。
        - **性能：** 通过优化算法得到增强。
        - **安全性：** 通过端到端加密得到加强。

        人味：
        更新改进了界面，通过优化算法加快了加载时间，
        并添加了端到端加密。

[填充词和回避模式]

    [填充短语]
        AI喜欢用冗长的表达替代简单的说法。

        常见替换：
        "为了实现这一目标" → "为了"
        "由于下雨的事实" → "因为下雨"
        "在这个时间点" → "现在"
        "在您需要帮助的情况下" → "如果需要帮助"
        "系统具有处理的能力" → "系统可以处理"
        "值得注意的是数据显示" → "数据显示"

    [过度限定]
        AI过度限定陈述，堆叠模糊词汇。

        AI味：
        可以潜在地可能被认为该政策
        可能会对结果产生一些影响。

        人味：
        该政策可能会影响结果。

    [通用积极结论]
        AI喜欢用模糊的乐观语句结尾。

        警惕词汇：
        未来看起来光明、激动人心的时代即将到来、
        继续追求卓越的旅程、
        向正确方向迈出的重要一步

        AI味：
        公司的未来看起来光明。
        激动人心的时代即将到来，
        他们继续追求卓越的旅程。

        人味：
        该公司计划明年再开设两个地点。

[交流痕迹]
    这些模式通常出现在AI直接对话的残留中，
    但有时也会渗透进生成的正文。

    [协作交流痕迹]
        警惕词汇：
        希望这对您有帮助、当然！、一定！、
        您说得完全正确！、您想要……、请告诉我

        这类痕迹在小说正文中绝对不应出现。

    [知识截止免责声明]
        警惕词汇：
        截至[日期]、根据我最后的训练更新、
        虽然具体细节有限/稀缺……、
        基于可用信息……

        同样不应出现在小说正文中。

    [谄媚语气]
        AI有时会在叙述中流露出过于积极、讨好的语气。

        AI味：
        好问题！您说得完全正确，这是一个复杂的话题。

        这类语气如果渗入角色对话或叙述者语气中，
        会严重破坏可信度。

[小说写作中的特殊AI味]
    以上模式是通用的AI痕迹。
    以下是小说创作中额外需要警惕的模式。

    [情感标签替代感受]
        AI倾向于直接命名情绪而非呈现情绪。

        AI味：
        她感到一阵深深的悲伤涌上心头，
        那种刻骨铭心的痛苦让她无法自已。

        人味：
        她站在窗前，手指在玻璃上划了一道。
        楼下的孩子在笑，她把窗户关上了。

        处理方式：参考 description-craft.md 中的心理外化策略。

    [角色声音同质化]
        AI写的不同角色说话方式趋于一致——
        都过于流畅、完整、得体。

        AI味对话：
        "我认为我们应该重新考虑这个计划的可行性。"
        "你说得对，我也觉得有必要进行更深入的分析。"

        人味对话：
        "这计划不行。"
        "哪不行？"
        "你自己不知道？"

        处理方式：参考 dialogue-writing.md 中的语言指纹策略。

    [叙述节奏过于均匀]
        AI生成的段落倾向于长度一致、结构一致，
        缺少节奏变化。

        AI味：
        连续五个段落都是80-100字，
        每段都是"描写+感受+行动"的结构。

        人味：
        段落长短交错。有时一段话只有一句。
        有时一个长段落沉浸在细节里，
        然后突然一个短句打断节奏。

    [隐喻堆砌]
        AI在一段话中使用多个不相关的隐喻。

        AI味：
        希望像火焰一样在他心中燃烧，
        又如潮水般汹涌澎湃，
        像蝴蝶破茧而出，展翅高飞。

        人味：
        他觉得胸口发热，像喝了一口烈酒。

        一个精准的隐喻胜过三个模糊的隐喻。

    [过度对称的句式]
        AI喜欢工整的对仗和对称结构。

        AI味：
        白天他是正义的化身，夜晚他是黑暗的使者。
        她的笑容温暖如春，她的眼神冷酷如冰。

        人味：
        白天他穿制服，晚上换一身不想让同事看见的衣服。
        她笑的时候眼睛弯成月牙，但你仔细看，
        那眼睛里什么都没有。

[个性与灵魂]
    去除AI模式只是一半的工作。
    还需要注入真实的写作个性。

    缺乏灵魂的写作迹象（即使技术上"干净"）：
    - 每个句子长度和结构相同
    - 没有观点，只有中立报道
    - 不承认不确定性或复杂感受
    - 没有幽默、没有锋芒、没有个性
    - 读起来像百科全书或新闻稿

    如何增加真实感：
    - 变化节奏：短句有力，长句铺开，混合使用
    - 承认复杂性：真实的人有矛盾的感受
    - 允许不完美：跑题、半成型的想法是人性的体现
    - 对感受要具体：不是"这令人担忧"，
      而是"半夜醒来想到这件事，睡不着了"
    - 有选择地呈现：真人不会面面俱到，
      会忽略一些东西，对另一些东西过度关注

[检查清单]
    写完后快速检查：
    - 连续三个句子长度相同？→ 打断其中一个
    - 段落都以简洁单行结尾？→ 变换结尾方式
    - 揭示前有破折号？→ 考虑删除
    - 解释了隐喻或比喻？→ 信任读者能理解
    - 使用了"此外""然而"等连接词？→ 考虑删除
    - 三段式列举？→ 改为两项或四项
    - 角色是否都说一样流畅的话？→ 检查语言指纹
    - 是否能大声朗读而不觉得别扭？→ 朗读测试
