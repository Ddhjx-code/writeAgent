# 质量标准定义

[概述]
    本文件为 review-checklist.md 中的十个审查维度提供详细的分级标准。
    每个维度定义了5分到1分各等级的具体表现，
    帮助审查时做出一致、可复现的判断。

    评分原则：
    - 以读者体验为核心标准，而非写作技巧的炫技程度
    - 关注"是否有问题"而非"是否完美"
    - 3分是合格线，意味着不影响正常阅读但有改进空间
    - 问题的严重程度取决于它是否让读者"出戏"

[维度一：叙事逻辑]

    5分 — 优秀：
    事件之间因果清晰，每个行动都有合理动机。
    信息揭示时机恰当，读者不会感到困惑或被欺骗。
    冲突的解决方式在此前有铺垫，回头看一切合理。

    4分 — 良好：
    整体逻辑通顺，有个别小处理可以更自然。
    如：某个角色的反应速度稍快，但不影响理解。

    3分 — 合格：
    主线逻辑成立，但有一到两处需要读者"脑补"才能说通。
    如：角色做了一个选择，动机可以推测但文中没有充分展示。

    2分 — 有问题：
    存在明显逻辑漏洞，读者会产生"为什么"的疑问。
    如：角色突然知道了不应该知道的信息，
    或冲突被一个从未提及的手段轻易解决。

    1分 — 严重问题：
    因果链断裂，事件之间缺乏逻辑连接。
    如：角色行为完全无法解释，时间线混乱，
    或出现自相矛盾的情节。

[维度二：人物一致性]

    5分 — 优秀：
    角色言行与设定高度一致，同时展现了细微的成长或变化。
    不同角色的声音清晰可辨，遮住名字也能猜出是谁。
    角色在压力下的反应符合其性格逻辑。

    4分 — 良好：
    角色整体一致，有个别台词或反应稍显偏离但可以接受。
    如：角色在某个场景中比平时话稍多，但不出戏。

    3分 — 合格：
    角色大体符合设定，但有一到两处让人觉得"这人不太会这样"。
    如：一个谨慎的角色突然变得冲动，但缺少触发事件。

    2分 — 有问题：
    角色行为与设定明显矛盾，或多个角色说话方式趋同。
    如：粗犷的军人说出了文绉绉的长句，
    或三个不同角色的对话语气几乎一样。

    1分 — 严重问题：
    角色完全脱离设定，像是换了一个人。
    或所有角色的声音完全无法区分。

[维度三：连贯性]

    5分 — 优秀：
    与前文衔接紧密自然。时间线、位置、情绪状态完全一致。
    前文的伏笔和悬念有呼应或推进。
    读者从上一章读到本章没有任何跳跃感。

    4分 — 良好：
    整体衔接通顺，有个别细节稍有出入但不影响阅读。
    如：上章提到角色左手受伤，本章有一处描写用了左手，
    但不是关键情节。

    3分 — 合格：
    主要衔接没问题，但有一到两个悬念被遗忘，
    或角色状态过渡不够自然。
    如：上章末尾角色情绪崩溃，本章开头恢复太快。

    2分 — 有问题：
    时间线或位置有明显矛盾，或重要悬念被遗忘。
    如：上章是深夜，本章开头变成了早晨但没有交代过渡。

    1分 — 严重问题：
    与前文严重脱节，像是两个不同的故事。
    多个悬念被遗忘，角色状态完全对不上。

[维度四：场景规划执行]

    5分 — 优秀：
    规划中的所有核心事件和情感目标都已实现。
    如有合理偏离，偏离带来了更好的效果。

    4分 — 良好：
    规划中的核心内容基本完成，有个别次要目标
    被弱化但不影响整体。

    3分 — 合格：
    核心事件已呈现，但情感基调或某些场景目标
    与规划有偏差。

    2分 — 有问题：
    遗漏了规划中的关键事件，或情感方向明显偏离规划。

    1分 — 严重问题：
    本章内容与场景规划严重不符，
    大部分规划内容未被执行。

[维度五：节奏与张力]

    5分 — 优秀：
    节奏变化丰富，紧张与缓解交替自然。
    开头有力，高潮有冲击力，结尾留有余韵。
    段落长度和信息密度有明显的有意变化。

    4分 — 良好：
    整体节奏合理，有个别段落稍显拖沓或仓促，
    但不影响阅读体验。

    3分 — 合格：
    节奏基本可以，但变化不够明显。
    如：大部分段落长度相近，缺少快慢对比。

    2分 — 有问题：
    节奏有明显问题。开头过于平淡，
    或中段严重拖沓，或高潮缺乏铺垫突然爆发。

    1分 — 严重问题：
    全章节奏平坦，没有起伏。
    或全章持续高压没有喘息空间，读者疲劳。

[维度六：对话质量]

    5分 — 优秀：
    每段对话都有明确功能，角色声音清晰可辨。
    对话中有潜台词和层次，动作与对话配合自然。
    对话节奏有变化，有打断、沉默、试探等真实感元素。

    4分 — 良好：
    对话整体有效，角色声音有区分。
    个别对话段落功能性稍弱，但不拖累整体。

    3分 — 合格：
    对话能推进情节，但缺少潜台词和层次。
    角色声音有一定区分，但不够鲜明。
    对话标签使用基本规范。

    2分 — 有问题：
    存在无功能的闲聊对话，或信息倾倒明显。
    角色说话方式趋同，难以区分谁在说话。
    对话标签过于花哨或使用副词过多。

    1分 — 严重问题：
    对话占据大量篇幅但不推进任何叙事任务。
    所有角色说话方式完全一样。
    存在角色互相说已知信息的严重信息倾倒。

[维度七：描写质量]

    5分 — 优秀：
    描写精准有力，用最少的笔墨激活具体的感官体验。
    心理状态通过动作、生理反应、环境互动外化呈现。
    隐喻新鲜精准，描写密度与节奏匹配。

    4分 — 良好：
    描写整体有效，有画面感和感官体验。
    个别处可以更精炼或更具体。

    3分 — 合格：
    描写能建立基本的场景感，但以视觉为主，
    其他感官较少涉及。偶有直接命名情绪的情况。
    隐喻中规中矩，没有明显问题也没有亮点。

    2分 — 有问题：
    描写过多导致拖沓，或过少导致场景缺乏画面感。
    频繁直接命名情绪（"她很悲伤""他非常愤怒"）。
    隐喻陈旧或堆砌。

    1分 — 严重问题：
    几乎没有有效描写，场景像剧本提纲。
    或描写大量堆砌形容词，读起来像说明文。
    隐喻混乱矛盾。

[维度八：去AI味]

    5分 — 优秀：
    文本读起来自然，有个性和声音。
    句式变化丰富，不存在AI典型模式。
    大声朗读不觉得别扭。

    4分 — 良好：
    整体自然，偶有一两处稍显工整或公式化，
    但不影响阅读体验。

    3分 — 合格：
    大部分文本自然，但能发现一些AI痕迹。
    如：三段式列举、个别高频AI词汇、
    段落结构偶尔过于对称。

    2分 — 有问题：
    AI痕迹明显。多处出现高频AI词汇密集使用、
    否定式排比、同义词循环、句式过于均匀等问题。
    读起来像是"机器写的，但还不错"。

    1分 — 严重问题：
    AI味浓重。大量填充短语、过度强调意义、
    宣传式语言、协作交流痕迹等。
    读者一眼就能判断是AI生成的。

[维度九：钩子与悬念]

    5分 — 优秀：
    开头在前三段内建立了强烈的吸引力。
    结尾悬念设置巧妙，类型与前几章有变化。
    旧悬念有回应，新悬念自然生长。
    悬念强度与当前叙事位置匹配。

    4分 — 良好：
    开头有吸引力，结尾有悬念。
    个别处可以更巧妙，但整体有效。

    3分 — 合格：
    开头不算平淡但也不够抓人。
    结尾有一定的翻页动力，但悬念力度一般。
    旧悬念基本有照顾到。

    2分 — 有问题：
    开头平淡，前几段缺少冲突或事件。
    结尾交代太完整，缺少翻页动力。
    或旧悬念被遗忘。

    1分 — 严重问题：
    开头让人没有读下去的欲望。
    结尾没有任何悬念，读者没有理由翻到下一章。
    多个旧悬念被遗忘且无新悬念提出。

[维度十：字数与密度]

    5分 — 优秀：
    字数在3000-5000字范围内。
    内容密度均匀合理，没有注水也没有过度压缩。
    场景数量和核心事件数量与字数匹配。

    4分 — 良好：
    字数基本达标，内容密度整体合理。
    个别段落稍显松散或紧凑，但不影响整体。

    3分 — 合格：
    字数达标但密度不均。
    如：某些场景展开充分，另一些明显被压缩。

    2分 — 有问题：
    字数不足3000或超过5000且有冗余。
    存在明显的注水段落或大段被跳过的场景。

    1分 — 严重问题：
    字数严重不足（低于2000）或严重超标（超过6000）。
    内容密度极度不均，大量无效内容或关键内容缺失。

[跨维度关联]
    某些问题可能同时影响多个维度，审查时注意：

    - 角色突然知道不该知道的信息
      → 同时影响"叙事逻辑"和"人物一致性"
    - 大段无功能描写
      → 同时影响"描写质量""节奏与张力""字数与密度"
    - 对话中的信息倾倒
      → 同时影响"对话质量"和"去AI味"
    - 遗忘旧悬念
      → 同时影响"连贯性"和"钩子与悬念"

    跨维度问题在报告中只需记录一次，
    标注它影响的所有维度即可。

[审查心态]
    审查的目的是帮助章节变得更好，而非找茬。

    注意事项：
    - 先看整体效果，再看细节问题
    - 优点也值得记录，帮助writer了解什么做得好
    - 问题描述要具体可操作，避免"感觉不太好"
    - 区分"必须修改"和"可以更好"的问题
    - 不要用自己的风格偏好替代质量标准
